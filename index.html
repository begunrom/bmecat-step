bmecat-step by begunrom

This is a Pentaho data integration step that can be used to generate XML files according to the BMECAT specification.

This is a Pentaho data integration step that can be used to generate XML files according to the BMECAT specification. For more information on the BMECAT standard visit http://www.bmecat.org/English/index.asp? For the moment we only support a subset of the BMECAT format.

Dependencies

This pdi step is tested against Pentaho data integration version 4.2-GA

Install

Install Pentaho data integration Copy the content of the zip file into the plugins\steps directory. Start PDI and look for a BMECAT group. Inside it you will find the BMECAT step. Open the bmecat-sample.ktr file and generate your first xml file in BMECAT format. In order to get your own data in, change the dummy data into real data, by replacing the data-grid steps with database input steps.
